# Installing {{site.data.keyword.wxbia_short}} on IBM Software Hub
{: #install-software}

Complete the following steps to install {{site.data.keyword.wxbia_short}} on IBM Software Hub.

{: #shortdesc}

## Procedure
{: #install-procedure}

If you are not an existing IBM Software Hub user, complete all of the following steps.

If you are an existing IBM Software Hub user but a new watsonx BI user, skip to [Preparing to install {{site.data.keyword.wxbia_short}}](#preparebi).

1. [Prepare to install IBM Software Hub](#prepareswh)

1. [Install IBM Software Hub](#installswh)

1. [Prepare to install {{site.data.keyword.wxbia_short}}](#preparebi)

1. [Install {{site.data.keyword.wxbia_short}}](#installbi)

1. [Complete the {{site.data.keyword.wxbia_short}} post-installation steps](#postinstallbi)

## Preparing to install IBM Software Hub
{: #prepareswh}

Before you install IBM Software Hub, read the [Planning](https://www.ibm.com/docs/en/software-hub/latest?topic=planning){: external} documentation. Make sure that your software, hardware, and storage meet the requirements for both IBM Software Hub and watsonx BI.

## Install IBM Software Hub
{: #installswh}

Complete the steps to [install IBM Software Hub](https://www.ibm.com/docs/en/software-hub/5.3.x?topic=installing#install__title__3){: external}.  You must complete 11 separate installation steps.

## Prepare to install watsonx BI
{: #preparebi}

Before you install watsonx BI, make sure that all prerequisites are met. The prerequisites include:
- System requirements

- Storage requirements

- Setting up a client workstation

- Installing the IBM Software Hub control plane

- Preparing your private container registry

- Installing GPU operators

- Installing Red Hat OpenShift AI

- Installing the required prerequisite services and models

For more information, see [Before you begin installing watsonx BI](https://www.ibm.com/docs/en/software-hub/5.3.x?topic=bi-installing#wxbi-install__title__3){: external}.

## Install {{site.data.keyword.wxbia_short}}
{: #installbi}

After all the prerequisites are met, [install {{site.data.keyword.wxbia_short}}](https://www.ibm.com/docs/en/software-hub/5.3.x?topic=bi-installing#wxbi-install__procedure__title__1){: external}.

## Complete the {{site.data.keyword.wxbia_short}} post-installation steps
{: #postinstallbi}

After you validate your {{site.data.keyword.wxbia_short}} installation, complete the following post-installation steps:

- [Create a service instance](https://www.ibm.com/docs/en/software-hub/5.3.x?topic=pis-creating-service-instance-1){: external}

- [Complete the post-installation setup](https://www.ibm.com/docs/en/software-hub/5.3.x?topic=setup-post-installation-watsonx-bi){: external}



## What to do next
{: #install-next-steps}

After IBM Software Hub and watsonx BI are both installed and set up, you are ready to begin using watsonx BI. To get started, [add users](https://www.ibm.com/docs/en/software-hub/5.3.x?topic=administering-roles-permissions#wxbi_roles__title__4) see [Creating metrics for conversations](/docs/watsonx-bi?topic=watsonx-bi-data_analysts_overview){: external}.
